# kawamoto-claude-plugins

Takeshi Kawamoto が作成した Claude Code プラグイン集。

## Plugins

| Plugin | Description |
|--------|-------------|
| [cri](./plugins/cri) | コーディング規約の自動抽出・統合（Continuous Rule Improvement） |
| [tf-docs-header](./plugins/tf-docs-header) | Terraform の `main.tf` に terraform-docs ヘッダを自動付与・更新 |

## Installation

このリポジトリをマーケットプレイスとして追加する。

```shell
/plugin marketplace add iret-kawamoto/kawamoto-claude-plugins
```

追加後、各プラグインを個別にインストールできる。詳細は各プラグインの README を参照。
